Реестр функций приложения

translateText Domain
Переводит текст с одного языка на другой с учетом контекста и форматирования.
text String
Исходный текст для перевода
sourceLanguage Language
Исходный язык текста
targetLanguage Language
Целевой язык перевода
Возвращает: TranslationResult
Результат перевода с форматированием и дополнительной информацией
Пример использования: 
val result = translateText(
    text = "Hello, world!",
    sourceLanguage = Language.ENGLISH,
    targetLanguage = Language.RUSSIAN
)
Зависимости:
TranslationService
TextFormatter
